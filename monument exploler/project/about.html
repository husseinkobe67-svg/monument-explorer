Who We Are

We are a team of five students — Hussein, Zeinab, Aya, Mhmd, and Carla — who worked hard and collaboratively on this project. Each team member contributed their time, effort, and ideas to design, develop, and present this website. Through teamwork and dedication, we combined our skills to create a meaningful and informative project.

Project Purpose

The main purpose of this project is to educate and inform users about historical monuments while developing our web development skills. We chose to focus on monuments in Istanbul to highlight their historical, cultural, and architectural importance. This project aims to present reliable information in a clear, organized, and visually appealing way. It also allowed us to practice teamwork, creativity, research, and time management while applying what we learned in class to a real-world digital project.

Tools & Technologies Used

This website was built using core web development technologies. HTML was used to structure the content and create the layout of each page. CSS was used to design the website, control colors, fonts, spacing, and ensure a clean and user-friendly appearance. JavaScript was used to add interactivity and enhance the user experience. By combining these tools, we were able to create a responsive, organized, and engaging website from scratch.
